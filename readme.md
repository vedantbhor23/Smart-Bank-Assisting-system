# 🏦 Smart Bank Assisting System

A comprehensive Android-based banking solution designed to simplify banking operations for both customers and administrators while enhancing security through facial authentication.

The system consists of separate applications for bank users and administrators, integrated with Firebase services and a dedicated face authentication module to provide a secure and user-friendly banking experience.

---

## 📱 Project Architecture

```
Smart-Bank-Assisting-System
│
├── 📂 Bank Admin
│     └── Admin application for bank staff
│
├── 📂 Bank User
│     └── Customer banking application
│
└── 📂 Face Authentication
      └── Face recognition & authentication module
```

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/vedantbhor23/Smart-Bank-Assisting-system-.git
```

### Open Project

1. Open Android Studio.
2. Select the required module.
3. Sync Gradle.
4. Configure Firebase using your own `google-services.json`.
5. Build and Run.
